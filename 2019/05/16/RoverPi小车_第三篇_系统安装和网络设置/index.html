激活SSH

从2016年11月的版本开始，Raspbian默认把SSH关闭．可以通过如下步骤手动激活SSH：

  1. 从Preferences菜单启动 Raspberry Pi Configuration
  2. 导航到 Interfaces 条目
  3. 选择 SSH旁边的Enabled
  4. 点击 OK
